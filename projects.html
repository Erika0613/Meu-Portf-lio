Meus Projetos

Card 1

O jogo do número secreto é um jogo em que o jogador deve tentar adivinhar um número específico. No caso do código que estamos estudando, o número secreto é o número 5. O jogador faz um chute, ou seja, escolhe um número, e o código verifica se o número escolhido é igual ao número secreto. Se o número for igual, o jogador recebe uma mensagem de parabéns. Caso contrário, o jogador recebe uma mensagem de que errou. O jogo é uma ótima forma de praticar condicionais e interação com o usuário em JavaScript.

Card 2

O Alura Midi é um instrumento musical de interface digital que será desenvolvido ao longo do curso. Ele é um projeto que utiliza HTML, CSS e JavaScript para criar uma experiência interativa de música. A ideia é que os alunos possam aprender a desenvolver essa interface utilizando JavaScript, explorando recursos atuais da linguagem para criar um instrumento musical dinâmico e interativo.

Card 3

O projeto Alura Fokus é inspirado na técnica Pomodoro, que visa aumentar a concentração e a produtividade. Nele, você aprenderá a manipular o DOM (Document Object Model) usando JavaScript. A técnica consiste em focar em uma atividade por um período de tempo determinado, como 25 minutos, e depois fazer uma pausa curta de 5 minutos ou uma pausa mais longa de 15 minutos.
